# Estoque Solidário

Sistema de controle de estoque desenvolvido em Java para uma creche beneficente, com acesso via celular e foco em qualidade de software.

## Objetivo do Projeto
Criar uma solução simples, gratuita e acessível para auxiliar instituições de caridade no controle de seus estoques, como alimentos, materiais de limpeza e itens infantis.

Além do impacto social, o projeto tem como objetivo demonstrar conhecimentos práticos na área de **Qualidade de Software (QA)**.

## Funcionalidades
- Cadastro de produtos
- Registro de entrada de produtos
- Registro de saída de produtos
- Consulta de estoque
- Interface responsiva (mobile)

## Tecnologias Utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Banco de dados H2
- HTML, CSS e Bootstrap
- Git e GitHub

## Qualidade e Testes
- Análise de requisitos
- Casos de uso
- Casos de teste manuais
- Testes de performance e stress com JMeter
- Identificação de gargalos
- Automação de testes (Selenium)

## Público-Alvo
Instituições beneficentes e creches sem recursos financeiros para aquisição de sistemas pagos.
