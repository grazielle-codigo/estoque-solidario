# Requisitos Funcionais — Estoque Solidário

## RF01 – Cadastrar produto
O sistema deve permitir cadastrar um produto informando nome e quantidade.

## RF02 – Listar produtos
O sistema deve permitir listar todos os produtos cadastrados no estoque.

## RF03 – Atualizar produto
O sistema deve permitir atualizar o nome e a quantidade de um produto existente, utilizando seu ID.

## RF04 – Deletar produto
O sistema deve permitir excluir um produto do estoque utilizando seu ID.
