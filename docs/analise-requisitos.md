# Análise de Requisitos – Sistema Estoque Solidário

## 1. Visão Geral
O sistema Estoque Solidário tem como objetivo permitir o controle simples e eficiente de estoque de uma creche beneficente, possibilitando o acesso por dispositivos móveis.

## 2. Usuários do Sistema
- Administrativo da creche

## 3. Requisitos Funcionais

RF01 – Cadastrar produtos no sistema  
RF02 – Registrar entrada de produtos no estoque  
RF03 – Registrar saída de produtos do estoque  
RF04 – Consultar a quantidade disponível de cada produto  

## 4. Requisitos Não Funcionais

RNF01 – O sistema deve ser acessível via navegador em celular  
RNF02 – O sistema deve ser gratuito  
RNF03 – Interface simples e intuitiva  
RNF04 – Tempo de resposta inferior a 2 segundos em consultas  

## 5. Restrições
- Não utilizar ferramentas pagas
- Utilizar linguagem Java
