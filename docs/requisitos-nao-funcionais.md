# Requisitos Não Funcionais — Estoque Solidário

## RNF01 – Usabilidade
O sistema deve ser simples e fácil de usar, podendo ser operado por pessoas sem conhecimento técnico.

## RNF02 – Desempenho
O sistema deve responder às requisições em até 2 segundos em condições normais de uso.

## RNF03 – Disponibilidade
O sistema deve estar disponível durante o horário de funcionamento da instituição.

## RNF04 – Segurança
O sistema não deve permitir acesso não autorizado às funcionalidades administrativas.

## RNF05 – Compatibilidade
O sistema deve funcionar corretamente em navegadores modernos e dispositivos móveis.
